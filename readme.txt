SAVINGS ACCOUNT PROGRAM

This is a simple Java program that allows users to manage a savings account. 
Users can check their balance, deposit money, and withdraw funds.

FEATURES:
- View account balance
- Deposit money
- Withdraw money (with sufficient balance check)
- Simple menu for easy navigation

TECHNOLOGIES USED:
- Java (Basic Console Application)
- Scanner Class for user input

HOW TO RUN THE PROGRAM:

Steps to Run:
1. Open a terminal or command prompt.
2. Compile the Java file:
   javac SavingsAccount.java
3. Run the program:
   java SavingsAccount
4. Follow the on-screen instructions to interact with the program.

Example Menu:
1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit
